# Getting Started

## Accessing the System

### For Customers

1. Open your web browser
2. Go to the website URL
3. The homepage will load automatically

### For Administrators

1. Go to the website URL
2. Add `/admin` to the URL (example: `website.com/admin`)
3. Enter your admin username and password
4. Click Login

## Creating a Customer Account

### How to Register

1. Click **Register** in the top navigation
2. Fill in the registration form:
   - Full Name (required)
   - Username (required)
   - Email Address (required)
   - Phone Number (required - Kenyan format: 07XX or 01XX followed by 8 digits)
   - Password (required)
3. Click **Create Account**
4. Check your email for verification (if enabled)

### Important Notes

- Phone number must be in Kenyan format
- Example valid numbers: `0712345678` or `012345678`
- Password should be at least 6 characters

## Logging In

1. Click **Login** in the navigation
2. Enter your username or email
3. Enter your password
4. Click **Sign In**
5. If login fails 5 times, you must wait 15 minutes

## Recovering Your Password

1. Click **Forgot Password** on the login page
2. Enter your registered email
3. Click **Send OTP**
4. Check your phone for a verification code
5. Enter the OTP code
6. Create a new password
7. Click **Reset Password**

# Glossary

## Technical Terms

### API (Application Programming Interface)
A set of rules that allows different software to communicate. The system uses APIs for mobile apps and integrations.

### Backend
The server-side of the website. Handles database, processing, and logic that users don't see.

### Cart
A virtual basket that holds products you want to buy. Persists until checkout or empty.

### Checkout
The process of completing a purchase. Includes entering delivery details and payment.

### Coupon
A code that gives a discount. Can be percentage or fixed amount off.

### CRUD
Create, Read, Update, Delete. The four basic operations for managing data.

### Dashboard
The main admin page. Shows statistics and quick overview of store performance.

### Database
Where all information is stored. Includes products, users, orders, and more.

### Delivery Zone
A geographic area with specific delivery fees and rules.

### Frontend
The part of the website users see. The customer-facing interface.

### Hero Section
The large banner at the top of a webpage. Used for promotions and key messages.

### Loyalty Points
Rewards earned by customers. Can be redeemed for discounts.

### M-Pesa
